Smart Paste

Paste a scorecard in any text format — WhatsApp message, notes app, photo-to-text. The parser reads it and fills every field. Review the parsed preview, then apply to the form.

Smart Paste — Format Guide
✅ Supported Batter Formats
Enoch 84 (36) c Adith b Sachin Monish 102 (45) b Shreyans Shivu 45* (20) not out Name | 84 | 36 | c X b Y Name 84 36 c X b Y
✅ Supported Bowler Formats
Shreyans 4-0-42-2 Sachin 4 42 2 Sachin | 4 | 42 | 2 Lakshya 3.4 38 1
✅ Innings Headers (optional)
1st Innings / First Innings 2nd Innings / Second Innings RCB Innings / RCB 320/6 (20 ov) BATTING / BOWLING (section markers)
✅ Result Lines (optional)
RCB won by 42 runs MI won by 3 wickets POTM: Enoch — 84(36) + 2/18 Extras: 8 Overs: 20
💡 Tips:
• Make sure the fixture teams are selected in Step 1 before pasting — the parser uses them to assign innings.
• If you only paste one innings, parse it, then paste the second innings separately and parse again.
• The parser auto-detects winners from run totals if no "won by" line is found.
• Names with * (e.g. Shivu 45*) are automatically marked not-out.
• After applying, always review the entries in the Quick Scorecard tab before saving.
